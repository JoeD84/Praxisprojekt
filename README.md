# Praxisprojekt #

## Übersetzen von Schrittmotorbefehlen
## Entwurf eines Hardwareübersetzers

### Hardware:

*	ATMEGA 324 A
*	STK500 Entwicklungsboard
* AVRISP mkII Programmer
* 4x20 LC-Display 
* VI-900 3D-Lasererfassungssystem

### Programmiersprache: 

* C

### Programmierumgebung:

* Eclipse mit AVRPlugin und AVRDude
* ~~AVR-Studio 5~~
